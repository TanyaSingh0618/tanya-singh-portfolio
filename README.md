## 🚀 How to Run Locally

You can preview and test this portfolio directly on your Mac:

### Option 1: Open Directly in Browser
Simply double-click `index.html` inside the `tanya-singh-portfolio` folder on your Desktop!

### Option 2: Run with Local Dev Server (Recommended)
If you have Python installed:
```bash
# Navigate to the project folder on your Desktop
cd /Users/tanyasingh/Desktop/tanya-singh-portfolio

# Start local HTTP server on port 8000
python3 -m http.server 8000
```
Then open your browser and visit: `http://localhost:8000`

## 🌐 Quick Publishing & Deployment Guide

To share this live URL with recruiters, colleagues, and your network right away, follow the step-by-step instructions inside **[`PUBLISHING_GUIDE.md`](./PUBLISHING_GUIDE.md)**.

### Fast Track — GitHub Pages (Free & Instant)
1. Create a new repository on your GitHub account (`https://github.com/TanyaSingh0618`) named `tanya-singh-portfolio` or `TanyaSingh0618.github.io`.
2. Push the files (`index.html`, `styles.css`, `script.js`, `enhanced_resume.md`) to your main branch from your Desktop folder:
   ```bash
   cd /Users/tanyasingh/Desktop/tanya-singh-portfolio
   git init
   git add .
   git commit -m "feat: initial release of Tanya Singh cloud-native & OR-Tools portfolio"
   git branch -M main
   git remote add origin https://github.com/TanyaSingh0618/TanyaSingh0618.github.io.git
   git push -u origin main
   ```
3. Go to **Repository Settings > Pages**, select `main` branch, and save!
4. Your live URL will be: `https://tanyasingh0618.github.io/` (or `https://tanyasingh0618.github.io/tanya-singh-portfolio/` if using the sub-path repo name).
